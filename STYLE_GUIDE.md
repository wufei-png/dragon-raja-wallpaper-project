# 风格总览与差异

## 1. 简笔画 / 涂鸦表情风

参考图属于这一类：白底、粗黑线、低细节、少量高饱和点缀色、情绪表达强、构图直接。

适合做：
- 梗图
- 角色状态图
- 原著名场面的反差版
- 小红书封面或系列番外

不适合做：
- 最震撼的龙王降临
- 高细节世界观展示
- 壁纸站主图

关键控制词：
- rough doodle
- hand-drawn sketch
- thick black messy lines
- white background
- minimal color accent
- deadpan expression

## 2. 动漫赛璐璐风

线条清晰，色块干净，阴影分层明确，角色辨识度强。适合人物情绪和剧情瞬间。

风险：
- 容易变成普通二次元角色图
- 龙和环境容易被弱化
- 表情好看但场面不够大

## 3. 日系动画电影感

比普通动漫更重视背景、空气感、光影和镜头语言。像电影关键帧，不像角色立绘。

适合：
- 东京夕阳
- 雨夜高架
- 深海探照灯
- 黑天鹅港雪夜
- 情绪悲剧场景

## 4. 黑暗都市奇幻

现代城市 + 超自然入侵。雨、霓虹、隧道、高架、地铁、警报灯、反常空间。

适合：
- 奥丁雨夜高架
- 北京地铁尼伯龙根
- 东京红井
- 卡塞尔学院夜战

## 5. 史诗概念艺术

最高细节与最强冲击力。重点不是人物脸，而是尺度、场景设计、龙形态、光影结构。

适合：
- 日本海沟龙类遗迹
- 青铜城
- 红井白王
- 王与王的天空战争
- 黑天鹅港龙骸

## 6. 游戏 CG 宣传图

更像游戏 key visual：动作、特效、角色姿态、武器、能量冲击明显，适合传播。

风险：
- 容易出现游戏 UI、技能特效过满、廉价手游感
- 需要明确禁止 UI 和 logo

## 7. 出版封面插画

更克制、更象征化，有留白，有中心意象，适合做系列封面。

## 8. 水墨神话风

可用于“龙”的东方神性表达，但《龙族》的核心不是传统国风神龙，需要保留现代 / 炼金 / 古文明异质感。

## 9. 黑白漫画分镜

强叙事、强冲击线、黑白高对比，适合做名场面分镜版。

## 风格选择建议

- 要“震撼壁纸”：优先史诗概念艺术、黑暗都市奇幻、游戏 CG 宣传图。
- 要“剧情氛围”：优先日系动画电影感、出版封面插画。
- 要“传播梗图”：优先简笔画 / 涂鸦表情风。
- 要“角色吸引力”：优先动漫赛璐璐风。
