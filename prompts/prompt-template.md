# Prompt 模板

## 中文结构版

```text
主题：{场景名称}
原著锚点：{卷名 / 场景摘要}
画幅：{16:9 / 21:9 / 9:16}

画面内容：
{地点、时间、天气、人物、龙、关键道具}

构图：
{前景 / 中景 / 背景}
{人物比例}
{龙的比例}
{镜头：超广角 / 低机位 / 俯视 / 远景 / 特写}

风格：
{从 styles/ 中复制风格片段}

光线与色彩：
{主色、辅色、光源、氛围}

必须出现：
- {must_have_1}
- {must_have_2}
- {must_have_3}

禁止：
{从 styles/00-base-negative.md 复制通用负面词}
{再加本场景负面词}
```
