# 负面词库

## 通用

```text
no text, no logo, no watermark, no signature, no subtitles, no game UI,
no health bars, no buttons, no random Chinese characters,
no copied official cover, no anime screenshot, no game screenshot,
no cute dragon, no pet dragon, no toy dragon, no generic western fire dragon,
no low resolution, no blurry, no deformed face, no extra limbs,
no crowded composition, no cheap mobile game poster
```

## 避免跑偏到仙侠

```text
no xianxia cultivators, no flying swords, no palace romance, no festive red-gold fantasy poster
```

## 避免跑偏到赛博朋克

```text
no cyberpunk interface, no holographic UI, no neon overload, no futuristic armor unless specified
```

## 避免跑偏到纯角色图

```text
no simple character portrait, no idol poster, no close-up face dominating the frame
```
