# Dragon-Raja-Inspired AI Wallpaper Project

> 非官方同人视觉策划项目。目标是把《龙族》原著关键场景转化为高清壁纸 / 海报级视觉图。
> 建议用途：个人学习、非商业分享、风格研究、提示词工程。不要声称“官方图”“正版海报”，不要复制官方封面、动画截图、游戏资产或 Logo。

## 项目目标

1. 以原著情节为锚点，不只做角色头像。
2. 同一个场景横向尝试多种视觉风格：
   - 简笔画 / 涂鸦表情风
   - 动漫赛璐璐风
   - 日系动画电影感
   - 黑暗都市奇幻
   - 史诗概念艺术
   - 游戏 CG 宣传图
   - 出版封面插画
   - 水墨神话风
   - 黑白漫画分镜
3. 输出三类规格：
   - 16:9 桌面壁纸
   - 21:9 超宽屏壁纸
   - 9:16 手机壁纸
4. 每张图记录：场景来源、原著一致性要点、prompt、负面词、版本结论、发布备注。

## 推荐流程

1. 在 `canon/scene-taxonomy.md` 里选择场景。
2. 在 `scenes/` 中打开对应“场景卡”。
3. 从 `styles/` 中选择一种风格卡。
4. 用 `prompts/prompt-template.md` 组合成最终 prompt。
5. 在 ChatGPT 网页版生图 / 局部修改。
6. 在 `outputs/<scene-id>/` 中记录结果，不一定把高清图纳入 git。
7. 用 `review/` 中的检查表做筛选。
8. 用 `publish/` 中的模板准备小红书、壁纸站或社交平台文案。

## 命名规则

```text
<scene-id>__<style-id>__<aspect-ratio>__v<version>.<ext>

示例：
004-deep-sea-dragon-ruins__epic-concept-art__16x9__v01.png
006-odin-rainy-viaduct__dark-urban-fantasy__21x9__v03.png
```

## 核心原则

- 人类要小，场景和龙要大。
- 龙不是宠物、坐骑或普通西幻喷火怪，而应像神祇、灾害、古生物、炼金造物。
- 不要生成文字、水印、UI、Logo。
- 不要直接照搬官方角色立绘、封面构图、动画截图或游戏宣传图。
- 原著情节只做简述和视觉转译，不在项目里保存大段原文。
