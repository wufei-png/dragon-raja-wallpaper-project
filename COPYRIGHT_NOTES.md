# 版权与发布注意

## 基本定位

本项目是非官方同人视觉策划与提示词工程，不是官方设定集、官方画集、官方动画图、官方游戏素材库。

## 建议

- 公开发布时标注：AI fan art / 非官方同人 / 灵感来自原著情节。
- 不要使用官方 Logo、书名题字、出版社封面元素、动画截图、游戏 UI。
- 不要直接复刻官方封面构图。
- 不要把图包商业销售为“龙族官方壁纸”。
- 不要在 prompt 里要求模型“完全照搬某官方封面”。
- 场景说明使用自己的短摘要，不保存长段原文。

## 可接受的项目记录方式

可以记录：
- 场景名称
- 卷名
- 简短情节摘要
- 视觉关键词
- 自己写的 prompt
- 自己的生成结果评价

不建议记录：
- 大段小说原文
- 扫描版官方画集
- 官方图的高清无授权搬运
