# 上传参考图：简笔画 / 涂鸦表情风

文件：`references/style-reference-doodle.jpg`

## 视觉特征

- 白色背景。
- 黑色粗线条，线条有手绘抖动感。
- 细节低，但表情直接。
- 只使用少量点缀色，本例是蓝色头发和黄色玫瑰。
- 文字框像随手画出的漫画框。
- 人物表情冷淡、疲惫、反差幽默。
- 适合做梗图、表情包、系列番外，不适合承担主视觉里的史诗尺度。

## 可迁移到本项目的方式

不要直接复制这个人物形象；迁移它的“风格语法”：

- 粗黑线
- 白底
- 低细节
- 死鱼眼 / 冷淡表情
- 少量高饱和色点缀
- 手写文字框
- 反差幽默

## 示例用法

把“红井白王苏醒”做成梗图版：

```text
白底粗线涂鸦，一个面无表情的少年站在巨大红色井口前，井里伸出一只极简白色龙爪，少年举着牌子写着“今天也不想拯救世界”，粗黑线，低细节，少量红色点缀，不要复杂背景。
```

壁纸正式版建议去掉文字框，避免生成乱码。
