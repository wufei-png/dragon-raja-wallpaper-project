# 上传参考图：简笔画 / 涂鸦速写风

文件：`references/style-reference-doodle.jpg`

## 视觉特征

- 白色背景。
- 黑色粗线条，线条有手绘抖动感。
- 细节低，但表情直接。
- 只使用少量点缀色，本例是蓝色头发和黄色玫瑰。
- 文字框像随手画出的漫画框，但本项目默认不用文字，避免乱码和语义跑偏。
- 人物表情冷淡、疲惫，画面有手绘即时感。
- 适合做同一场景的极简风格对照，不适合承担唯一主视觉里的史诗尺度。

## 可迁移到本项目的方式

不要直接复制这个人物形象；迁移它的“风格语法”：

- 粗黑线
- 白底
- 低细节
- 冷淡、克制或悲剧性的表情
- 少量高饱和色点缀
- 手绘边框感
- 简化但不改写场景

## 示例用法

把“红井白王苏醒”做成简笔速写版：

```text
白底粗线涂鸦，一个极小的少年剪影站在巨大红色竖井边缘，井中升起极简白色龙王轮廓，远处用几条线表示东京夜景和直升机光点，粗黑线，低细节，少量红色和冷白点缀，严肃悲剧氛围，不要文字。
```

壁纸正式版建议去掉文字框，避免生成乱码。
