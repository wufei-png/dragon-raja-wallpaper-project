# 原著场景分类

## S 级：最适合做震撼壁纸

1. 夔门白帝城水下青铜城
   - 核心视觉：长江深水、夔门 / 白帝城旧影、青铜迷宫、龙文穹顶、蛇脸人雕像、青铜与火权能。
   - 适合风格：史诗概念艺术、日系动画电影感、水墨神话风。

2. 摩尼亚赫号暴雨屠龙
   - 核心视觉：三峡暴雨、摩尼亚赫号拖船 / 军舰伪装、氙灯、直升机探照灯、龙侍参孙、水下炸弹火柱。
   - 适合风格：游戏 CG 宣传图、黑暗都市奇幻、漫画分镜。

3. 北京地铁尼伯龙根 / 芬里厄
   - 核心视觉：福寿岭 / 100号站 / 第零站、检修小车、废弃地铁、无限隧道、芬里厄黄金眼、地下王国。
   - 适合风格：黑暗都市奇幻、史诗概念艺术、动漫电影感。

4. 日本海沟高天原 / 神葬所
   - 核心视觉：迪里雅斯特号、八千米深海、鸟居、中央高塔、黑色风铃、巨大锁链、岩浆河、列宁号。
   - 适合风格：史诗概念艺术、电影概念设定图。

5. 红井 / 赫尔佐格夺取白王权能
   - 核心视觉：多摩川山区红井、白丝、茧、绘梨衣悲剧线索、赫尔佐格 / 新白王、直升机、天谴前兆。
   - 适合风格：史诗概念艺术、游戏 CG 宣传图、黑暗都市奇幻。

6. 0号高速雨夜 / 奥丁尼伯龙根
   - 核心视觉：0号 / 10号高速错位、暴雨高架、迈巴赫、八足马、蓝斗篷、冈格尼尔、白色命运线。
   - 适合风格：黑暗都市奇幻、日系动画电影感、黑白漫画分镜。

## A 级：适合剧情氛围图

7. 黑天鹅港 / 列宁号甲板古龙骨
   - 核心视觉：西伯利亚无名港、极地冻土、实验基地、列宁号、甲板古龙骨、白裙雷娜塔、零号逃亡。
   - 适合风格：动画电影感、史诗概念艺术、出版封面插画。

8. 东京逃亡 / 路明非与绘梨衣的小怪兽之旅
   - 核心视觉：东京商业街 / 游乐园、天空树、红发绘梨衣、路明非、玩偶 / 明信片、远处红井或白王阴影。
   - 适合风格：动漫赛璐璐、动画电影感、出版封面插画。

9. 卡塞尔学院夜色
   - 核心视觉：学院、钟楼、雨夜、龙文、学生剪影、暗处龙影。
   - 适合风格：黑暗都市奇幻、动漫电影感。

## 使用原则

- S 级场景优先用作项目主视觉。
- A 级场景适合补充角色情绪和世界观。
- 每个场景都可生成“同内容不同风格”的风格矩阵。
- 生成前优先读取对应 `scenes/*.md` 的“内容锁定”和“基础 Prompt”；不要只凭本分类表扩写。
