# 原著场景分类

## S 级：最适合做震撼壁纸

1. 三峡青铜城 / 水底龙宫
   - 核心视觉：水下青铜迷宫、龙文、古代龙类文明、火与水的反差。
   - 适合风格：史诗概念艺术、日系动画电影感、水墨神话风。

2. 摩尼亚赫号暴雨屠龙
   - 核心视觉：暴雨、江面、研究船、探照灯、龙影、炮火。
   - 适合风格：游戏 CG 宣传图、黑暗都市奇幻、漫画分镜。

3. 北京地铁尼伯龙根 / 芬里厄
   - 核心视觉：废弃地铁、无限隧道、巨龙眼睛、地下王国。
   - 适合风格：黑暗都市奇幻、史诗概念艺术、动漫电影感。

4. 日本海沟深处龙类遗迹
   - 核心视觉：深潜器、黑色海流、海底古城、巨龙骨架、深海探照灯。
   - 适合风格：史诗概念艺术、电影概念设定图。

5. 红井 / 白王苏醒 / 东京毁灭危机
   - 核心视觉：血红竖井、白色龙王、东京夜空、警报、直升机、末日感。
   - 适合风格：史诗概念艺术、游戏 CG 宣传图、黑暗都市奇幻。

6. 奥丁雨夜高架 / 尼伯龙根
   - 核心视觉：暴雨、高架桥、黑马、长枪、路灯无限重复、现实断裂。
   - 适合风格：黑暗都市奇幻、日系动画电影感、黑白漫画分镜。

## A 级：适合剧情氛围图

7. 黑天鹅港
   - 核心视觉：雪夜实验基地、龙骸、火光、逃亡、命运起点。
   - 适合风格：动画电影感、史诗概念艺术、出版封面插画。

8. 东京逃亡 / 绘梨衣与路明非
   - 核心视觉：夕阳、东京街头、红发少女、少年、远处龙影或命运阴影。
   - 适合风格：动漫赛璐璐、动画电影感、出版封面插画。

9. 卡塞尔学院夜色
   - 核心视觉：学院、钟楼、雨夜、龙文、学生剪影、暗处龙影。
   - 适合风格：黑暗都市奇幻、动漫电影感。

## 使用原则

- S 级场景优先用作项目主视觉。
- A 级场景适合补充角色情绪和世界观。
- 每个场景都可生成“同内容不同风格”的风格矩阵。
