## 场景核对结论

### 001 夔门白帝城水下青铜城

- 白帝城线索应绑定长江三峡 / 夔门 / 青铜与火之王诺顿，不应误解为白王。
- 关键视觉应包括水下青铜城、龙文穹顶、蛇脸人雕像、青铜机关、潜水员尺度对比。
- 画面不应画成普通海底龙宫、仙侠宫殿或热带海底。

### 002 摩尼亚赫号暴雨屠龙

- 摩尼亚赫号在原文中有拖船外形与军舰级能力的反差。
- 关键视觉应包括深夜三峡暴雨、氙灯、直升机 / 探照灯、龙侍参孙、黑色背脊、巨口、勾齿、水下炸弹与水火柱。
- 不应把对象画成普通海怪或完整西幻龙王。

### 003 北京地铁尼伯龙根 / 芬里厄

- 第二卷地铁尼伯龙根不是泛泛的废弃地铁；福寿岭、100号站、第零站、检修小车、备用轨道等细节能显著提高贴合度。
- 芬里厄视觉上要保留大地与山之王的巨大体量，幼稚心智只能做微弱彩蛋，不能萌化。
- 可用黄金眼、黑蟒般长舌 / 巨影、隧道与岩壁融合来表现。

### 004 日本海沟高天原 / 神葬所

- 深潜器应使用迪里雅斯特号，海面支援为须弥座；主体场景在日本海沟八千米深处。
- 高天原视觉锚点优先级高于泛化“龙骨遗迹”：鸟居、中央高塔、黑色风铃、巨大金属锁链、岩浆河、列宁号沉船、胚胎 / 尸守线索。
- 避免画成普通深海科幻基地或珊瑚遗迹。

### 005 红井 / 赫尔佐格夺取白王权能

- 红井应是多摩川山区一带的巨型红色深井 / 藏骸之井，不是东京市中心火山口。
- 重点是赫尔佐格利用绘梨衣等人的血统夺取白王权能，画面要有白丝、茧、圣骸、黑色直升机、天谴前兆和东京危机。
- 绘梨衣线索应悲剧化、远景化，避免血腥实验特写和战斗海报化。

### 006 0号高速雨夜 / 奥丁尼伯龙根

- 关键视觉是 0号高速 / 10号高速错位、雨夜高架、迈巴赫、八足马、暗金铠甲、蓝斗篷、冈格尼尔、白色命运线。
- 奥丁不是普通骑士；画面不应跑到中世纪战场。
- 龙性以尼伯龙根规则、风雨、黑影随从和现实折叠表现。

### 007 黑天鹅港 / 列宁号甲板古龙骨

- 黑天鹅港在西伯利亚北部无名港 / 北冰洋语境下，不是普通雪山基地。
- 关键视觉包括极地冻土、锈铁码头、实验基地、列宁号核动力破冰船、甲板古龙骨、苍白骨翼、右眼金焰、雷娜塔与零号逃亡。
- 不应画成温馨圣诞雪景或普通恐龙化石展。

### 008 东京逃亡 / 路明非与绘梨衣的小怪兽之旅

- 这组画面应保留东京日常与末日阴影的反差：商业街、游乐园、天空树、台场 / 惠比寿等地点，玩偶、明信片、冰淇淋、小皮箱等道具。
- 绘梨衣是红发少女，路明非以 Sakura 身份陪她短暂逃离。
- “小怪兽”是情绪和关系，不是要画成真的可爱怪兽；远处危机应只作为红井 / 白王 / 冷白云影提示。

## 风格核对结论

- `references/style-reference-doodle.jpg` 应被理解为“同一内容的简笔 / 涂鸦速写渲染”，不是剧情改编。
- 每个 scene 的 doodle prompt 必须保留同一地点、同一事件、同一人物比例、同一龙 / 超自然元素。
- 禁止加对白框、段子文字、文字贴纸或无关小剧场。
