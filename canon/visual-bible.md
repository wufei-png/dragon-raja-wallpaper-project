# 视觉圣经

## 龙的统一设计语言

《龙族》壁纸里的龙不建议画成普通西幻喷火龙。更合适的方向：

- 神祇感：不是怪物，而是世界规则本身。
- 灾害感：像海啸、地震、暴雨、火山爆发一样不可谈判。
- 古文明感：鳞片、骨骼、青铜、炼金纹路、龙文互相融合。
- 非宠物化：不要坐骑、不要萌化、不要卡通表情。
- 体量压迫：龙可以只露出一只眼、一段脊骨、一只爪子，但尺度必须巨大。

## 人物比例

- 震撼场景：人物占画面 1%–8%。
- 剧情场景：人物占画面 20%–45%。
- 简笔画 / 涂鸦速写风：沿用同一场景的人物比例，不因为风格变成夸张大头或文字图。

## 色彩体系

- 青铜城：深绿、蓝黑、青铜金、熔融橙。
- 摩尼亚赫：黑蓝、雷电白、探照灯黄、血色少量点缀。
- 地铁尼伯龙根：暗黄灯光、黑灰隧道、龙眼金。
- 日本海沟：深海蓝黑、探照灯白、古城冷绿、龙骨灰白。
- 红井白王：血红、冷白、东京夜蓝、警报橙。
- 奥丁高架：雨夜黑、路灯钠黄、冷蓝反光、银白长枪。
- 黑天鹅港：雪白、钢铁灰、爆炸橙、龙骸黑影。
- 东京逃亡：东京暖橙、绘梨衣红发、游乐园彩灯、夜蓝、远处冷白 / 深红阴影。

## 构图原则

- 远景优先：先建立场景尺度。
- 前景放人类或装备，用来制造比例差。
- 中景放关键建筑 / 船 / 高架 / 地铁站。
- 背景放龙影 / 龙骨 / 龙王 / 裂缝。
- 保留负空间，便于做壁纸。

## 同内容不同风格

- 所有风格共享同一份内容锁定：地点、时间、核心事件、人物比例、龙的出现方式和主要色彩不能变。
- 风格只改变线条、材质、光影、细节密度和渲染方法。
- doodle-sketch 是严肃的低细节手绘速写：白底、粗黑线、少量点缀色、同样的构图和道具；不要加段子、对白框、文字贴纸或无关小剧场。
- 如果某一风格导致场景专名消失，例如摩尼亚赫号变成普通游轮、迪里雅斯特号变成普通潜水员、奥丁变成普通骑士，这张图应判定为不合格。
