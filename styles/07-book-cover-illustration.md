# style_id: book-cover-illustration
# 中文名：出版封面插画

## 视觉目标

克制、象征、留白、中心意象明确。适合做系列封面，不一定追求爆炸特效。

## Prompt 片段

```text
refined fantasy book cover illustration, symbolic composition, elegant negative space,
central visual metaphor, painterly detail, restrained color palette,
mysterious atmosphere, no typography, no title text
```

## 负面词补充

```text
no busy action scene, no poster text, no commercial logo, no cheap paperback look
```

## 构图建议

允许更抽象：一个井口、一只龙眼、一把长枪、一座水下青铜门。
