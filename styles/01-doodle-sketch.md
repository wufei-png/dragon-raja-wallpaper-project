# style_id: doodle-sketch
# 中文名：简笔画 / 涂鸦速写风

## 视觉目标

像随手画出来的严肃场景速写：低细节、粗黑线、白底、少量点缀色，保留同一画面的地点、人物、龙/超自然存在和构图关系。

## Prompt 片段

```text
rough hand-drawn doodle, thick messy black lines, white background, minimal details,
serious minimalist scene, same composition and same scene content, sparse color accents,
intentionally rough sketch, flat white space, clear silhouettes, no written joke
```

## 负面词补充

```text
no realistic rendering, no complex background, no highly detailed painting, no glossy 3D,
no polished anime key visual, no meme text, no speech bubble, no parody, no comedic gag
```

## 构图建议

保留场景卡里的内容锁定。可以把复杂背景压缩成少量线条和色块，但不能删掉关键地点、人物比例、龙/超自然存在方式和道具。默认无文字、无对话框。
