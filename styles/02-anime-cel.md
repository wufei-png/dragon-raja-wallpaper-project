# style_id: anime-cel
# 中文名：动漫赛璐璐风

## 视觉目标

清晰线稿、干净色块、分层阴影、角色面部可读性高。

## Prompt 片段

```text
clean anime cel shading, sharp line art, expressive character design,
dramatic but readable lighting, crisp color blocks, high-quality anime key visual,
detailed eyes, controlled background detail
```

## 负面词补充

```text
no chibi, no moe overload, no school comedy mood, no generic idol poster
```

## 构图建议

人物可占 30%–50%，龙和场景必须保留空间，否则会变成普通角色图。
