# style_id: dark-urban-fantasy
# 中文名：黑暗都市奇幻

## 视觉目标

现代城市中出现超自然裂缝。真实城市材质 + 神秘灾害感。

此风格限定用于现代城市 / 地铁 / 高架 / 东京 / 奥丁尼伯龙根等场景。不要用于水下青铜城、深海高天原、极地黑天鹅港这类古文明或自然环境主导的场景；这些场景优先使用 `epic-concept-art`、`donghua-key-visual`、`book-cover-illustration` 或 `ink-mythic`。

## Prompt 片段

```text
dark urban fantasy, rain-soaked modern city, neon reflections,
supernatural anomaly, ominous atmosphere, grounded realism,
cinematic lighting, wet asphalt, massive unseen presence, oppressive scale
```

## 负面词补充

```text
no medieval fantasy village, no bright heroic fantasy, no cyberpunk UI overload
```

## 构图建议

现代物体必须存在：高架、地铁、路灯、警报、直升机、城市天际线。
超自然元素要像入侵现实，而不是完全架空世界。
