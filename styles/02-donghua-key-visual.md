# style_id: donghua-key-visual
# 中文名：国漫宣传主视觉

## 参考图

`references/224328-16799282089d75.jpg`

只迁移视觉语法，不复制原图构图、Logo、角色、城市或龙形。

## 视觉目标

国漫 / 动画宣传海报式主视觉：半写实数字绘、清晰大透视、强天气和天光、现代或现实锚点、一个极小人物作为尺度、巨大龙影或超自然天象压在背景中。它吸收原先动漫赛璐璐、动画电影感和游戏 CG 宣传图里适合壁纸传播的部分，但不保留角色大头、纯动画截帧或游戏 UI / 战斗特效倾向。

## Prompt 片段

```text
donghua promotional key visual, Chinese animated series poster look,
semi-realistic anime digital painting, dramatic wide-angle perspective,
modern-world scale anchor, tiny human silhouette for scale,
colossal dragon-shaped cloud or supernatural dragon shadow integrated with the sky or environment,
polished commercial key art, cinematic sunset or storm light,
clear readable composition, strong atmosphere, high wallpaper appeal
```

## 负面词补充

```text
no logo, no title text, no copied Dragon Raja poster, no official artwork recreation,

```

## 构图建议

优先用“大环境 + 极小人物 + 巨大龙影 / 天象”的传播图结构。人物可以是屋顶、船舷、潜水灯、车旁或站台边缘的剪影，用现实物体建立尺度；龙或超自然存在不要直接画成完整西幻怪物，而应像云、风暴、水体、青铜机关或城市阴影的一部分。
