# 通用负面词

```text
no text, no logo, no watermark, no game UI, no subtitles, no frame border unless requested,
no official book cover recreation, no copied anime screenshot, no copied game asset,
no cute pet dragon, no generic western fire-breathing dragon, no toy-like dragon,
no low resolution, no blurry face, no extra limbs, no malformed hands,
no cluttered composition, no random Chinese characters, no unreadable typography
```

## 中文说明

- 禁止文字：AI 很容易生成乱码。
- 禁止 UI：尤其游戏 CG 风容易出现血条、技能按钮。
- 禁止普通西幻龙：本项目的龙应更像古老神祇、灾害、炼金生命、巨大古生物。
- 禁止官方复刻：避免版权和风格混淆。
