# 原著一致性检查表

## 基础检查

- [ ] 同一场景不同风格共用同一份内容锁定。
- [ ] 场景地点正确。
- [ ] 时间 / 天气大体正确。
- [ ] 核心事件没有被改成无关情节。
- [ ] 龙的存在方式符合该场景。
- [ ] 人物不是无关角色。
- [ ] 简笔画 / 涂鸦速写风没有改成文字牌、对白框或无关小剧场。
- [ ] 没有混入其他作品元素。
- [ ] 没有官方 Logo 或复刻官方封面。

## 视觉解释是否合理

- [ ] 即使没看过小说，也能看出场景冲突。
- [ ] 看过小说的人能识别情节锚点。
- [ ] 人类与龙的尺度关系合理。
- [ ] 风格变化没有破坏内容一致性。
