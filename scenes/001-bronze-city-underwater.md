# 夔门白帝城水下青铜城

## 原著锚点

《龙族 I：火之晨曦》的水下主线发生在长江三峡夔门、白帝城一带。原文把“白帝”与金属之王、青铜与火之王诺顿联系起来，不是白王线索；青铜城内部有龙文穹顶、蛇脸人雕像、青铜机关和龙类文明遗迹。叶胜、酒德亚纪等人的潜入行动与摩尼亚赫号水面支援构成同一组事件。

## 场景摘要

深暗江水下，一座青铜机械迷宫从夔门和白帝城旧影中显形。龙文穹顶、蛇脸侍从雕像、青铜门和炼金机关像仍在缓慢运转，潜入者的探照灯只照亮极小一角。画面重点不是“华丽龙宫”，而是人类误入金属之王墓室 / 城邦时面对古老文明和青铜与火权能的压迫。

## 内容锁定

- 地点：长江三峡夔门 / 白帝城水下青铜城，不是海洋。
- 时间 / 环境：深水、低能见度、冷暗江水，青铜机关中有少量熔火般暖光。
- 核心事件：人类潜入青铜城，发现龙文穹顶、蛇脸人雕像和青铜与火之王诺顿相关遗迹。
- 人物与比例：1-3 个潜水员或人类剪影极小，占画面 1%-8%。
- 龙 / 超自然存在方式：不必完整画出诺顿；以龙侍阴影、巨大黄金瞳、青铜鳞片、龙文机关或沉睡王座/骨殖意象暗示。
- 主要色彩：蓝黑、深绿、氧化青铜、少量熔融橙金。

## 必须出现

- 水下环境和悬浮颗粒。
- 青铜建筑、青铜门或机械迷宫结构。
- 龙文 / 炼金纹路。
- 龙文穹顶或宏大的青铜天井感。
- 蛇脸人雕像，带古代礼制 / 侍臣感。
- 潜水灯、人类剪影或通讯线作为尺度对比。
- 青铜与火之王相关的苏醒迹象：巨大龙影、黄金瞳、青铜茧、龙侍或蛇脸侍从雕像之一。

## 可选出现

- 白帝城 / 夔门的水下旧城暗示。
- 七宗罪匣、古代甲胄、环首刀、蛇脸人雕像。
- 熔融金属般的暖光从青铜缝隙里透出。
- 探照灯锥形光束切开水体。

## 禁止出现

- 普通海底宫殿或仙侠龙宫。
- 珊瑚、热带鱼、旅游潜水感。
- 可爱龙、坐骑龙、普通西幻喷火龙。
- 过亮童话色调。
- 明确复刻官方封面、动画截图或游戏素材。

## 龙的设计语言

这里的龙性应该和青铜结构融合：像沉睡的金属生命、古老神祇和炼金机关的共同体。可以只露出一只巨大黄金瞳、一片青铜鳞甲、一段爪影或龙侍黑影，让观众感觉整座城本身就是龙王权能的延伸。

## 构图建议

- 16:9 / 21:9：超广角远景，前景放潜水员和探照灯，中景放青铜门与蛇脸人雕像，背景放龙文穹顶、巨大龙影或黄金瞳。
- 9:16：竖向青铜门 / 穹顶结构，人物在底部，龙文和龙眼在画面上方深处。
- doodle-sketch：同一构图简化成白底粗线，青铜城用几何线框，潜水员用小黑影，龙眼用金色点缀，不加文字。
