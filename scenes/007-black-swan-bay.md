# 黑天鹅港 / 列宁号甲板古龙骨

## 原著锚点

黑天鹅港位于西伯利亚北部无名港附近，临近北冰洋，是赫尔佐格旧实验体系的起点。原文关键视觉不是普通雪地实验室，而是灰白永久冻土、锈铁码头、苏联红军哨兵感、圣诞夜撤离 / 毁灭计划、列宁号核动力破冰船、甲板上的巨大古龙骨、被切走的龙眼和最终站起的苍白骨翼龙影。雷娜塔与零号的逃亡线让这组画面带有童年噩梦和命运起点的质感。

## 场景摘要

北极雪夜，黑天鹅港实验基地在风雪、探照灯和爆炸火光中崩塌。锈铁码头旁停着列宁号，甲板上横陈的巨大古龙骨像一座被冰封的神殿；白裙小女孩和小男孩的剪影在钢结构之间逃离。画面要冷、残酷、实验性强，但不能做成温馨圣诞或普通灾难片。

## 内容锁定

- 地点：西伯利亚北部无名港 / 黑天鹅港，靠近北冰洋，不是城市港口。
- 时间 / 环境：雪夜、极地冻土、风雪、寒冷蓝白光和爆炸橙光。
- 核心事件：黑天鹅港毁灭，列宁号带走龙族禁忌物，雷娜塔 / 零号逃亡。
- 人物与比例：白裙雷娜塔和零号 / 逃亡者剪影很小，占画面 1%-10%。
- 龙 / 超自然存在方式：列宁号甲板上的古龙骨、苍白骨翼、被唤醒的右眼金焰或黑蛇般古龙影子。
- 主要色彩：雪白、极夜蓝、钢铁灰、锈红、爆炸橙、龙眼金。

## 必须出现

- 极地雪夜、冻土或锈铁码头。
- 黑天鹅港实验基地 / 冷战设施感。
- 列宁号破冰船或大型船体轮廓。
- 巨大古龙骨 / 苍白骨翼 / 甲板上的龙类遗体。
- 爆炸火光、探照灯或撤离混乱。
- 小小逃亡者剪影，最好有白裙女孩暗示雷娜塔。

## 可选出现

- 苏联风格红星、旧军用卡车、铁丝网、岗亭，但不要喧宾夺主。
- 教堂屋顶、走廊、通风管道或圣诞晚会反差暗示。
- 液氮容器、铁柜、基因样本箱等实验道具。
- 黑蛇般古龙影子掠过屋顶或甲板。

## 禁止出现

- 温馨圣诞村、童话雪景、明亮度假港口。
- 普通恐龙化石展览。
- 现代豪华邮轮。
- 把雷娜塔 / 零号画成成年战斗主角。
- 过度血腥实验特写。

## 龙的设计语言

黑天鹅港的龙性应该像旧时代罪证：古龙骨不是博物馆标本，而是能在雪夜和火光里重新站起的神话遗骸。可以突出苍白骨翼、右眼金焰、甲板上龙骨和船体比例，让它既像死物又像马上复活。
