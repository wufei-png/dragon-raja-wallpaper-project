# 红井 / 赫尔佐格夺取白王权能

## 原著锚点

《龙族 III：黑月之潮（下）》的红井不是普通火山口。它是多摩川山区一带的巨型深井 / 藏骸之井，蛇岐八家与卡塞尔都把它视为白王线索的终点。原文关键视觉包括红色井水、地下赤色河流、鬼齿龙蝰等亚种威胁、赫尔佐格以绘梨衣等人为容器 / 营养夺取白王权能、白丝与茧、黑色直升机、天谴和东京毁灭危机。

## 场景摘要

东京近郊山中的红井像被打开的伤口，深红水汽和警报光从井口涌出。井底不是可爱的白龙，而是人类阴谋、白王圣骸和绘梨衣悲剧叠在一起的灾难现场：赫尔佐格在白丝和茧中完成进化，路明非来得太晚，天空中的直升机和远处东京夜景都显得渺小。画面要有末日、神性和悲剧，但不要把重点变成血腥猎奇。

## 内容锁定

- 地点：东京近郊多摩川山区的红井 / 藏骸之井，不是城市正中央裂开的火山口。
- 时间 / 环境：夜晚、暴雨或湿冷山谷，红色井水 / 水汽、警报灯、直升机光束。
- 核心事件：赫尔佐格借绘梨衣等人的血统夺取白王权能，红井成为白王复苏危机的中心。
- 人物与比例：路明非或执行者剪影很小；若画绘梨衣，只作为远处被白丝 / 茧包裹的悲剧中心，不做血腥特写。
- 龙 / 超自然存在方式：白王权能以白色圣骸、白丝、巨大茧、冷白龙影、鳞翼或天空灾厄暗示。
- 主要色彩：深红、冷白、东京夜蓝、警报橙、黑色雨夜。

## 必须出现

- 巨型红井 / 藏骸之井，深红水汽或赤色水面。
- 白丝、茧或圣骸感的白色结构。
- 赫尔佐格 / 新白王的冷白灾厄影子，或正在成形的白色龙王剪影。
- 绘梨衣悲剧线索：远处小小红发少女轮廓、白丝包裹的人影、红色小皮箱或散落玩具之一。
- 黑色直升机、探照灯、东京远景或山中设施，体现现代围剿和城市危机。

## 可选出现

- 鬼齿龙蝰 / 龙族亚种在红水中形成细长黑影。
- 天谴前兆：高空冷光、云层裂口、达摩克利斯之剑式坠落感。
- 路明非抱起绘梨衣的剪影，但必须远景处理。
- 井壁上的管线、输血装置、铁架和蛇岐八家设施残骸。

## 禁止出现

- 普通火山喷发、岩浆湖、城市地震裂缝。
- 可爱白龙、圣洁宠物龙、英雄式驯龙画面。
- 过度血腥、医学特写、裸露实验体特写。
- 把绘梨衣画成战斗海报中心位。
- 游戏 UI、技能特效、明显官方封面复刻。

## 龙的设计语言

白王权能应该像冷白灾害和神性污染：白丝、茧、骨翼、苍白鳞光、天空坠落物和红井里的深红水汽共同构成“新白王正在诞生”的感觉。这里的恐怖来自命运和仪式，不来自普通怪兽造型。
