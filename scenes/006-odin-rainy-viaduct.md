# 奥丁雨夜高架 / 尼伯龙根

## 场景摘要

雨夜高架桥进入异常空间，路灯无限重复，黑马与持枪神影从雨幕中逼近。重点是现实被折叠后的恐惧。

## 必须出现

- 暴雨
- 高架桥
- 无限路灯 / 车灯
- 黑马或神秘骑影
- 长枪剪影

## 可选出现

- 积水反光
- 被雨模糊的城市
- 远处龙形云影
- 破碎现实裂缝

## 禁止出现

- 中世纪战场
- 阳光草原
- 普通骑士插画
- 夸张游戏 UI

## 龙的设计语言

这里的龙意象可弱化为神话阴影或云层龙形；主压迫源是奥丁般的骑影和长枪。

## 推荐风格

黑暗都市奇幻、日系动画电影感、漫画分镜、出版封面插画。
