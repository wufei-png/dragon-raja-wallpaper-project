# 东京逃亡 / 路明非与绘梨衣的小怪兽之旅

## 原著锚点

《龙族 III：黑月之潮（下）》里，绘梨衣主动想“出去玩”，与路明非在追捕和末日阴影下短暂逃离。原文视觉锚点不是普通恋爱约会，而是红发少女、Sakura 称呼、玩偶箱、商业街和游乐园、东京迪士尼 / 台场 / 惠比寿等城市日常、天空树、红色情人旅馆、“我们都是小怪兽”的互相确认，以及远处红井和白王计划的阴影。

## 场景摘要

东京的日常灯光短暂包住两个被命运追赶的人。路明非和红发绘梨衣站在天台、商业街或游乐园边缘，手里可能有玩偶、冰淇淋、明信片或购物袋；远处天空树和城市霓虹很温柔，但云层里隐约有红井 / 白王危机的冷白阴影。画面要温柔、孤独、短暂，不要变成纯甜恋爱海报。

## 内容锁定

- 地点：东京城市日常场景，可选天台、商业街、游乐园 / 迪士尼外缘、台场或天空树附近。
- 时间 / 环境：傍晚、雨后夜景或游乐园灯光，温暖前景与冷色远景对照。
- 核心事件：路明非带绘梨衣逃离蛇岐八家控制，体验普通人的东京。
- 人物与比例：路明非与红发绘梨衣可以占画面 20%-45%；两人姿态克制，不做偶像海报。
- 龙 / 超自然存在方式：远处云层、城市反光或天空冷光暗示红井 / 白王 / 命运阴影，不画实体龙压过人物。
- 主要色彩：东京暖橙、游乐园彩灯、绘梨衣红发、夜蓝、远处冷白 / 深红阴影。

## 必须出现

- 少年路明非与红发绘梨衣。
- 东京城市锚点：天空树、商业街、游乐园灯光、台场摩天轮或地铁站口之一。
- 日常道具：玩偶、明信片、冰淇淋、购物袋、小皮箱或轻便旅行包之一。
- 远处危机暗示：红色云影、白色龙影、直升机光束、红井方向的冷光之一。
- 温柔但注定失去的气氛。

## 可选出现

- 绘梨衣白裙或简单外出装，红发被风吹起。
- 路明非使用 Sakura 这一身份的暗示，但不要出现文字。
- “小怪兽”只作为画面情绪：两个孤独的人被城市照亮，不要画成真的可爱怪兽。
- 自动贩卖机、商场橱窗、游乐园气球、长椅等待冰淇淋。

## 禁止出现

- 纯恋爱甜图、婚纱照、偶像宣传海报。
- 把绘梨衣画成战斗女王或魔法少女。
- 直接加文字、对白框、明信片可读内容。
- 让实体巨龙占满天空，压掉日常场景。
- 过度性感化或年龄感错误。

## 龙的设计语言

这张的龙性应该很轻：危险在远处，像云层、玻璃反光、红色天光或冷白光斑。真正的主体是“两个小怪兽短暂假装普通人”的悲剧，而不是龙王战斗本身。

## 构图建议

- 16:9：两人在前景或中景，东京夜景 / 天空树在远处，红井方向的红白云影压在天边。
- 21:9：横向展开商业街或游乐园灯光，两人偏左，右侧留城市负空间做壁纸。
- 9:16：人物在下方，天空树 / 摩天轮 / 云影在上方，形成温柔和危机的上下对照。
- doodle-sketch：同一画面简化成两个人、红发、东京塔线、玩偶和远处红云，不加文字。
